Como funciona este projeto?

HAProxy está sendo usado como balanceador de carga. Ele recebe requisições e distribui para 3 instâncias Flask:

  • app1
  • app2
  • app3

Você pode testar os algoritmos:

▶ Round Robin: localhost:8080
Distribui as requisições em sequência (1 → 2 → 3 → 1…)

▶ Least Connections: localhost:8081
Envia para o servidor menos ocupado.

Objetivo

Esta demonstração mostra visualmente qual servidor está processando sua requisição. Recarregue a página várias vezes para ver o balanceamento funcionando.
